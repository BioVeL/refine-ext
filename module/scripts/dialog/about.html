BioVeL Extension
The BioVeL extension augments already existing functionality provided by Open (Google) Refine with a focus on biodiversity data.
The primary goal of the extension is to provide a toolkit to improve and enhance Data Quality, with three main categories including :
  • Data Quality Checks
  • Data Transformation
  • Data Extension
More information regarding the BioVeL project can be found at http://www.biovel.eu/
This extension is part of the more comprehensive Data Refinement Workflow. Details related to the workflow including installation instructions, user guides, tutorials, etc. can be found here.
